# 好物购买系统 数据库表结构文档

> 数据库：MySQL 8.0
> 字符集：utf8mb4
> 排序规则：utf8mb4_unicode_ci

---

## 一、用户表 `t_user`

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK AUTO_INCREMENT | 用户ID |
| username | VARCHAR(50) | NOT NULL UNIQUE | 用户名 |
| password | VARCHAR(100) | NOT NULL | 密码（BCrypt加密） |
| phone | VARCHAR(20) | UNIQUE | 手机号 |
| email | VARCHAR(100) | UNIQUE | 邮箱 |
| avatar | VARCHAR(255) | | 头像URL |
| status | TINYINT | NOT NULL DEFAULT 1 | 状态：0=禁用，1=正常 |
| create_time | DATETIME | NOT NULL | 注册时间 |
| update_time | DATETIME | NOT NULL | 更新时间 |
| last_login_time | DATETIME | | 最后登录时间 |

```sql
CREATE TABLE t_user (
  id            BIGINT       NOT NULL AUTO_INCREMENT PRIMARY KEY,
  username      VARCHAR(50)  NOT NULL UNIQUE COMMENT '用户名',
  password      VARCHAR(100) NOT NULL COMMENT '加密密码',
  phone         VARCHAR(20)  UNIQUE COMMENT '手机号',
  email         VARCHAR(100) UNIQUE COMMENT '邮箱',
  avatar        VARCHAR(255) COMMENT '头像URL',
  status        TINYINT      NOT NULL DEFAULT 1 COMMENT '0=禁用 1=正常',
  create_time   DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
  update_time   DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  last_login_time DATETIME   COMMENT '最后登录时间'
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='用户表';
```

## 四、店铺表 `t_shop`

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK AUTO_INCREMENT | 店铺ID |
| merchant_id | BIGINT | NOT NULL FK | 所属商家ID |
| name | VARCHAR(100) | NOT NULL | 店铺名称 |
| logo | VARCHAR(255) | | 店铺Logo |
| description | TEXT | | 店铺简介 |
| address | VARCHAR(255) | | 店铺地址 |
| phone | VARCHAR(20) | | 联系电话 |
| status | TINYINT | NOT NULL DEFAULT 1 | 0=禁用，1=正常 |
| audit_status | TINYINT | NOT NULL DEFAULT 0 | 0=待审核，1=审核通过，2=审核拒绝 |
| audit_remark | VARCHAR(255) | | 审核备注 |
| rating | DECIMAL(3,1) | DEFAULT 5.0 | 综合评分 |
| sales_count | INT | DEFAULT 0 | 总销量 |
| create_time | DATETIME | NOT NULL | 创建时间 |
| update_time | DATETIME | NOT NULL | 更新时间 |

```sql
CREATE TABLE t_shop (
  id           BIGINT        NOT NULL AUTO_INCREMENT PRIMARY KEY,
  merchant_id  BIGINT        NOT NULL COMMENT '所属商家ID',
  name         VARCHAR(100)  NOT NULL COMMENT '店铺名称',
  logo         VARCHAR(255)  COMMENT '店铺Logo',
  description  TEXT          COMMENT '店铺简介',
  address      VARCHAR(255)  COMMENT '店铺地址',
  phone        VARCHAR(20)   COMMENT '联系电话',
  status       TINYINT       NOT NULL DEFAULT 1 COMMENT '0=禁用 1=正常',
  audit_status TINYINT       NOT NULL DEFAULT 0 COMMENT '0=待审核 1=通过 2=拒绝',
  audit_remark VARCHAR(255)  COMMENT '审核备注',
  rating       DECIMAL(3,1)  DEFAULT 5.0 COMMENT '综合评分',
  sales_count  INT           DEFAULT 0 COMMENT '总销量',
  create_time  DATETIME      NOT NULL DEFAULT CURRENT_TIMESTAMP,
  update_time  DATETIME      NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_merchant_id (merchant_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='店铺表';
```

## 六、商品表 `t_product`

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK AUTO_INCREMENT | 商品ID |
| shop_id | BIGINT | NOT NULL FK | 所属店铺ID |
| category_id | BIGINT | NOT NULL FK | 分类ID |
| name | VARCHAR(200) | NOT NULL | 商品名称 |
| description | TEXT | | 商品描述 |
| price | DECIMAL(10,2) | NOT NULL | 销售价格 |
| original_price | DECIMAL(10,2) | | 原始价格 |
| stock | INT | NOT NULL DEFAULT 0 | 库存数量 |
| main_image | VARCHAR(255) | NOT NULL | 主图URL |
| images | TEXT | | 商品图片组（JSON数组） |
| specs | TEXT | | 规格参数（JSON） |
| status | TINYINT | NOT NULL DEFAULT 0 | 0=下架，1=上架 |
| audit_status | TINYINT | NOT NULL DEFAULT 0 | 0=待审核，1=通过，2=拒绝 |
| audit_remark | VARCHAR(255) | | 审核备注 |
| sales_count | INT | DEFAULT 0 | 销量 |
| rating | DECIMAL(3,1) | DEFAULT 5.0 | 评分 |
| create_time | DATETIME | NOT NULL | 创建时间 |
| update_time | DATETIME | NOT NULL | 更新时间 |

```sql
CREATE TABLE t_product (
  id            BIGINT        NOT NULL AUTO_INCREMENT PRIMARY KEY,
  shop_id       BIGINT        NOT NULL COMMENT '所属店铺ID',
  category_id   BIGINT        NOT NULL COMMENT '分类ID',
  name          VARCHAR(200)  NOT NULL COMMENT '商品名称',
  description   TEXT          COMMENT '商品描述',
  price         DECIMAL(10,2) NOT NULL COMMENT '销售价格',
  original_price DECIMAL(10,2) COMMENT '原价',
  stock         INT           NOT NULL DEFAULT 0 COMMENT '库存',
  main_image    VARCHAR(255)  NOT NULL COMMENT '主图URL',
  images        TEXT          COMMENT '商品图片组JSON',
  specs         TEXT          COMMENT '规格参数JSON',
  status        TINYINT       NOT NULL DEFAULT 0 COMMENT '0=下架 1=上架',
  audit_status  TINYINT       NOT NULL DEFAULT 0 COMMENT '0=待审核 1=通过 2=拒绝',
  audit_remark  VARCHAR(255)  COMMENT '审核备注',
  sales_count   INT           DEFAULT 0 COMMENT '销量',
  rating        DECIMAL(3,1)  DEFAULT 5.0 COMMENT '评分',
  create_time   DATETIME      NOT NULL DEFAULT CURRENT_TIMESTAMP,
  update_time   DATETIME      NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_shop_id (shop_id),
  INDEX idx_category_id (category_id),
  INDEX idx_status (status),
  FULLTEXT INDEX idx_name (name)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='商品表';
```

## 七、购物车表 `t_cart`

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK AUTO_INCREMENT | 购物车ID |
| user_id | BIGINT | NOT NULL FK | 用户ID |
| product_id | BIGINT | NOT NULL FK | 商品ID |

```sql
CREATE TABLE t_cart (
  id          BIGINT       NOT NULL AUTO_INCREMENT PRIMARY KEY,

  product_id  BIGINT       NOT NULL COMMENT '商品ID',
```

## 表关系说明

```
t_user          ←── t_address      (一对多：用户拥有多个收货地址)
t_user          ←── t_cart         (一对多：用户购物车)
t_user          ←── t_order        (一对多：用户订单)
t_user          ←── t_favorite     (一对多：用户收藏)
t_user          ←── t_review       (一对多：用户评价)

t_merchant      ──→ t_shop         (一对一：商家拥有店铺)
t_shop          ←── t_product      (一对多：店铺商品)
t_shop          ←── t_order        (一对多：店铺订单)

t_product       ←── t_cart         (多对多：通过购物车关联)
t_product       ←── t_order_item   (快照关联)
t_product       ←── t_favorite     (多对多：通过收藏关联)
t_product       ←── t_review       (一对多：商品评价)
```
